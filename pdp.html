Políticas de Privacidade - Kash Bot

Ao usar o Kash Bot, você concorda com todos os Termos de Uso.

1. Informações que Coletamos

1.1 Dados do Usuário

Dados do Usuário incluem, mas não se limitam a, IDs de Usuário, IPs de Usuário, e-mails de usuário, fotos de perfil, nomes de usuário e tags de usuário (“discriminador”). A equipe Klee utiliza os Dados do Usuário como meio de personalizar sua experiência e ajudar você de qualquer maneira possível. O uso pode incluir, mas não se limita a, informações de depuração para comandos como "sugerir" ou informações de suporte para identificar um usuário, uso em embeds de mensagens, e registros sobre o usuário e uso de comandos.

1.2 Dados da Guilda

Dados da Guilda incluem, mas não se limitam a, IDs de Guilda, ícones de Guilda, nomes de Guilda, contagem de membros, funções, canais e funções possuídas por um membro. Os Dados da Guilda são um meio de personalizar ainda mais sua experiência e ajudar você de qualquer maneira que os Dados do Usuário não sejam úteis. O uso pode incluir, mas não se limita a, informações de depuração para comandos como "sugerir" ou informações de suporte para identificar uma guilda, uso em embeds de mensagens, e registros sobre o uso de comandos. Além disso, os desenvolvedores podem usar os Dados da Guilda de qualquer maneira que precisem, desde que não os usem para fins maliciosos. Os desenvolvedores se reservam o direito de gerar um convite para uma guilda a fim de fornecer suporte ou determinar se a guilda está abusando de nossos bots.

1.3 Uso e Armazenamento de Dados do Usuário e Dados da Guilda

A equipe Klee se reserva o direito de usar e armazenar Dados do Usuário e Dados da Guilda de qualquer maneira necessária para a funcionalidade de nossos bots. A equipe Klee não usará Dados do Usuário ou Dados da Guilda para fins maliciosos, mas apenas quando forem necessários para comandos ou recursos específicos de nossos bots. Os Dados do Usuário e da Guilda serão armazenados apenas em situações específicas em que forem necessários para a persistência de um recurso. Isso inclui, mas não se limita a, moeda do usuário, registro de eventos, listas negras e acesso a benefícios específicos.

2. Como Utilizamos as Informações

Utilizamos as informações para fornecer o serviço do Kash Bot a você, personalizando e melhorando nossos serviços. Também poderemos entrar em contato com você caso necessário. A equipe Klee se reserva o direito de remover e excluir permanentemente seu conteúdo ou dados do bot, com ou sem aviso, por qualquer motivo ou sem motivo.
