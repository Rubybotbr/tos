Políticas de Privacidad - Kash Bot

Al usar Kash Bot, aceptas todos los Términos de Uso.

1. Información que Recopilamos

1.1 Datos del Usuario

Los Datos del Usuario incluyen, entre otros, IDs de Usuario, IPs de Usuario, correos electrónicos de usuario, fotos de perfil, nombres de usuario y etiquetas de usuario ("discriminador"). El equipo Kash Team utiliza los Datos del Usuario como un medio para personalizar tu experiencia y ayudarte de cualquier manera posible. El uso puede incluir, entre otros, información de depuración para comandos como "sugerir" o información de soporte para identificar a un usuario, uso en mensajes embebidos y registros sobre el usuario y uso de comandos.

1.2 Datos de la Guilda

Los Datos de la Guilda incluyen, entre otros, IDs de Guilda, iconos de Guilda, nombres de Guilda, conteo de miembros, roles, canales y roles poseídos por un miembro. Los Datos de la Guilda son un medio para personalizar aún más tu experiencia y ayudarte de cualquier manera que los Datos del Usuario no sean útiles. El uso puede incluir, entre otros, información de depuración para comandos como "sugerir" o información de soporte para identificar una guilda, uso en mensajes embebidos y registros sobre el uso de comandos. Además, los desarrolladores pueden usar los Datos de la Guilda de cualquier manera que necesiten, siempre y cuando no los usen para fines maliciosos. Los desarrolladores se reservan el derecho de generar una invitación a una guilda con el fin de brindar soporte o determinar si la guilda está abusando de nuestros bots.

1.3 Uso y Almacenamiento de Datos del Usuario y Datos de la Guilda

El equipo Kash Team se reserva el derecho de usar y almacenar los Datos del Usuario y Datos de la Guilda de cualquier manera necesaria para la funcionalidad de nuestros bots. El equipo Kash Team no usará los Datos del Usuario o Datos de la Guilda para fines maliciosos, sino solo cuando sean necesarios para comandos o características específicas de nuestros bots. Los Datos del Usuario y de la Guilda se almacenarán solo en situaciones específicas en las que sean necesarios para la persistencia de una función. Esto incluye, entre otros, moneda de usuario, registro de eventos, listas negras y acceso a beneficios específicos.

2. Cómo Utilizamos la Información

Utilizamos la información para proporcionar el servicio de Kash Bot, personalizando y mejorando nuestros servicios. También podremos contactarte si es necesario. El equipo Kash Team se reserva el derecho de eliminar y eliminar permanentemente tu contenido o datos del bot, con o sin aviso, por cualquier motivo o sin motivo.
